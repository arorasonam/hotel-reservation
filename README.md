# 🏨 Hotel Reservation System

A robust, full-stack web application designed to streamline hotel operations, from guest room discovery to administrative booking management. Built with the **Laravel** framework, this system ensures secure transactions, real-time availability, and a seamless user experience.

---

## 🚀 Key Features

### **For Guests**
* **Smart Room Search:** Filter rooms by category, price range, and date availability.
* **Instant Booking:** Seamless reservation flow with automated email confirmations.
* **User Accounts:** Manage profiles, view booking history, and download invoices.
* **Secure Payments:** Integrated payment gateway support (Stripe/PayPal).

### **For Administration**
* **Dynamic Dashboard:** Real-time stats on total bookings, revenue, and pending check-ins.
* **Room Management:** Full CRUD for rooms, including image uploads and pricing tiers.
* **Booking Control:** Centralized view to approve, cancel, or modify guest reservations.
* **Occupancy Calendar:** Visual representation of room status (Available, Booked, Maintenance).

---

## 🛠️ Tech Stack

* **Backend:** PHP 8.2+ | Laravel 12
* **Frontend:** Blade Templates | Tailwind CSS or Bootstrap 5
* **Database:** MySQL / PostgreSQL
* **State Management:** Alpine.js or Livewire (optional)
* **Tools:** Vite, Composer, NPM
